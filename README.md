# Siddhant Sahai — Portfolio Website

Simple one-page portfolio built with HTML & CSS to showcase skills and projects.

---

## Features

- Responsive single-page layout
- Home & About sections
- Skills progress bars (C, Python, HTML/CSS, AI/ML basics)
- Projects showcase
- Contact section with form and social links
- Clean and minimal design

---

## Files

index.html   → Main website file  
style.css    → All styling for the website  
assets/      → (Optional) Folder for images and media  

---

## How to Run the Project

1. Download or clone the project.
2. Keep `index.html` and `style.css` in the same folder.
3. Open `index.html` in any web browser.
4. The website will load locally on your system.

---

## Customization

- Change profile name → Edit `index.html`
- Replace profile photo → Update image `src` path
- Edit skills → Modify skill bars in HTML/CSS
- Update projects → Add or remove project cards in HTML
- Adjust styles → Edit `style.css`

---

This portfolio website is made to:
- Practice frontend development skills.
